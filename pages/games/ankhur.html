Ank'hur
UnityC#Android
View on itch.io Play Store
Ank'hur is a puzzle game made from a board of sliding blocks. The goal of the game is to get each basic block into a target square - the player can tilt the board one of four directions and different blocks will slide in different ways.
  • Five sets of five levels, each set introducing a new type of block
  • Create your own levels and upload them for others to play! The block list is serialised as a string and uploaded as a highscore to dreamlo
  • Extensive use of Unity custom inspectors to facilitate debugging and level creation
  • Discord Rich Presence
I've also ported Ank'hur to Android and it's now available on the Play Store! The UI has had an overhaul as well as optimisations to make it suitable for mobile.
