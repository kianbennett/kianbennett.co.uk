Lute Hero
UnityC#
View on itch.io
Lute Hero is an rpg/rhythm game in a medieval setting. It was made with Unity/C# and the art style was inspired by Final Fantasy VI. Some notable features include:
  • Detailed rhythm input system for playing notes of a song
  • Turn-based combat
  • Inventory with equippable and consumable items
  • Multiple towns, each with their set of townsfolk with individual daily routines (pathfinding with Aron Granberg's A* Pathfinding Project)
  • Song chart parser allowing for custom songs to be loaded into the game (cross compatible with custom Guitar Hero charts)
The song recordings are by Jon Sayles.
